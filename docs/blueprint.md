# **App Name**: StockPilot

## Core Features:

- Product Search and Filtering: Enable users to quickly find products using a search bar and category filters.
- Product Table with Status Indicators: Display product information in a responsive table with color-coded status indicators (In Stock/Out of Stock).
- CSV Import/Export: Allow users to import product data from a CSV file and export the entire product catalog to a CSV file.
- Inline Editing: Enable inline editing of product details within the table, with save and cancel options.
- Inventory Change History: Track and display a history of inventory changes for each product, including date, old stock, new stock, and user.
- Add New Product: Button to add new products. Form implementation is optional.
- Track Updates: Log stock updates.

## Style Guidelines:

- Primary color: A deep blue (#3F51B5) to convey trust and reliability.
- Background color: A very light blue (#E8EAF6) to provide a calm backdrop.
- Accent color: A vibrant purple (#9C27B0) to highlight important actions and elements.
- Body and headline font: 'Inter', a sans-serif font, is used for the user interface.
- Use clear, minimalist icons from a library like Material Icons for actions and status indicators.
- Employ a responsive layout with clear visual hierarchy, using grid or flexbox for consistent alignment.
- Incorporate subtle animations for loading states and transitions to enhance user experience.
